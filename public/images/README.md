## Image Guidelines

- Use WebP format for better performance when possible
- Provide fallbacks in JPEG/PNG format
- Optimize images for web (max width 1920px for hero images)
- Include alt text for accessibility
- Use descriptive file names
- Maintain consistent brand colors and style

## Temporary Placeholders

Until final images are available, the following placeholders can be used:
- Use gradient backgrounds with brand colors
- Use solid color blocks with text overlays
- Use icon-based placeholders for missing images
